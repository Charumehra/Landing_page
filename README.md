# Infocera IT Landing Page

Infocera IT Landing Page is a modern, responsive marketing website prototype built for an IT services firm. It focuses on polished UI/UX interactions, mobile-first behavior, and layered visual design using modern frontend tooling.

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Lucide React icons
- React Router DOM (available)

## Features

- Responsive navigation with desktop multi-level dropdowns and mobile accordion menu
- Hero + services sections with animated gradients and backgrounds
- Testimonials, statistic highlights, and trust indicators
- Contact CTA form layout and interactive cards
- Sticky header scroll effects
